Projects
=====

The Project page is where you can add, edit, and delete your QGIS Projects.

Usage
------------

To add a Project, click Projects in the left menu


   .. image:: images/Projects-1.png

Click the "Add Project" button at top as show below:

   .. image:: images/Projects-2.png


Give the Project a name and use the dropdowns to select the desired value for each field.

Click Save.

 .. image:: images/Projects-5.png

Your Project has now been added

   .. image:: images/Projects-6.png

The Project info icon at right displays Project information.

   .. image:: images/Projects-8.png

Enter the required fields as shown below.
