
Quick Start
=============

In the Quick Start below, we'll use the Wizard to quickly synchronize a new QGIS Project.

Each step of this processed can be configured per project using the Database, Service, and Daemon menus.

1. Log In
------------

Once logged in, click on the Administration link at top as shown below:

  .. image:: images/Quick-Start-1.png


2.  Click the Setup Wizard.
------------

  .. image:: images/Quick-Start-2.png

2.  Select QFieldCloud, Mergin Maps, ArcGIS Survey123 or Self-Hosted.
------------

  .. image:: images/Quick-Start-3.png

3.  Enter your username and password and click Next.
------------

  .. image:: images/Quick-Start-4.png

  
4.  Click Connect
------------

  .. image:: images/Quick-Start-5.png

5.  Select the QGIS Project to sync.  Optionally, you can change the sync interval
------------

  .. image:: images/Quick-Start-6.png

5.  Click Create
------------

  .. image:: images/Quick-Start-8.png

6.  You are ready to start Synchronization.  Click
------------

  .. image:: images/Quick-Start-9.png

7.  On the Service page, click Start
------------

  .. image:: images/Quick-Start-9b.png

8.  Your QGIS Project is now being synchronized to PostGIS
------------

  .. image:: images/Quick-Start-11.png


As mentioed above, each step of the above processed can be set to a particular configuration using the menu tabs at left.
