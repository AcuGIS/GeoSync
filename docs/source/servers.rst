Providers
=====

The Provider page is where you add your QFieldCloud, Mergin Maps, and ArcGIS Survey123 server(s)

The applies whether the instance(s) is self-hosted or uses the Provider directly.

You can add an unlimited number of Providers

Usage
------------

On the left menu, click Providers

   .. image:: images/Providers.png

Click the "Add New" button at top right.

   .. image:: images/Providers-2.png

Give the entry a name, select Provider type (QField, Mergin Maps, or Survey123)

Pay attention to the url entered.  

   .. info::  
         Urls follow the same patterns whether self-hosted or provider hosted

Enter the required information and click Save

   .. image:: images/Providers-3.png

The Provider is now registered and you can now select Project(s) to synchronize with the Projects menu or Wizard

   .. image:: images/Providers-4.png

Actions
------------

The Actions sections allows you to:

1. View Admin Password

2. Edit Provider information

3. Delete Provider
